# STM32-based-RS485-Bus-Remote-Fish-Pond-Management-System
基于stm32的RS485总线远程鱼塘管理系统
作者：宋楠，辅助：杨建海，李思雨。
基于stm32的RS485总线远程鱼塘管理系统，该项目包含一个酸碱传感器，一个水质传感器，一个水温度传感器，并且通过485总线进行远距离传输，实现实时管理和控制。

Author: Song Nan, Assisted by: Yang Jianhai, Li Siyu. Based on the STM32 RS485 bus remote fish pond management system, this project includes a pH sensor, a water quality sensor, and a water temperature sensor, and uses the 485 bus for long-distance transmission to achieve real-time management and control.
